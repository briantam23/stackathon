# Air Quality Index App

## Setting up

### Dependencies

* [Express](https://expressjs.com)
* [EJS](https://ejs.co)

### Installation

* `npm install` (or `yarn install`)
* `npm start`
* open up [localhost:3000](http://localhost:3000) in a web browser


* also deployed to [Heroku](https://stackathon-btam.herokuapp.com)
